## Gatsby

To use MDX with Gatsby, use gatsby-plugin-mdx.

First, scaffold a new Gatsby 2.0 or greater site and install the `gatsby-plugin-mdx` plugin.

> `gatsby new gatsby-site`

> `cd gatsby-site`

> `yarn add gatsby-plugin-mdx @mdx-js/mdx@latest @mdx-js/react@latest`

Then add `gatsby-plugin-mdx` to your `gatsby-config.js` in the plugins section.

```c
module.exports = {
  siteMetadata: {
    title: `My Ambitious Project`
  },
  plugins: [`gatsby-plugin-mdx`]
}
```

Finally, add an `.mdx` file in the `src/pages` directory. It "Just Works".

```c
# My first MDX Page

some awesome content
```

For more documentation on programmatically creating pages with Gatsby, see the [Gatsby MDX docs](https://www.gatsbyjs.com/docs/how-to/routing/mdx/).

## Paragraph

You can use the paragraph syntax by typing inside .mdx file without using any symbol

```c
...
Lorem ipsum dolor sit amet consectetur adipisicing elit. Recusandae explicabo nulla beatae. Dolor id totam incidunt natus et atque rem consequatur dolorum quaerat, numquam tempora, minima quos tempore aut. Nam?
...
```

**And it will create something like this**:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Recusandae explicabo nulla beatae. Dolor id totam incidunt natus et atque rem consequatur dolorum quaerat, numquam tempora, minima quos tempore aut. Nam?

## Italic

```c
*Lorem ipsum dolor sit amet consectetur adipisicing elit.*

or

_Lorem ipsum dolor sit amet consectetur adipisicing elit._
```

**And it will create something like this**:

> **using the \*\* symbol** : _Lorem ipsum dolor sit amet consectetur adipisicing elit._

> **using the \_ symbol** : _Lorem ipsum dolor sit amet consectetur adipisicing elit._
